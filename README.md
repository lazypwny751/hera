# Hera
alternative package manager written in bash 5.0.17 developing for [bychan multi tool](https://github.com/ByCh4n/BCHackTool) and [konteynir]()

# Installation
```
git clone https://github.com/ByCh4n-Group/hera.git
cd hera
bash configure.sh
sudo make install
```
